# 22OrbitMigration (Next.js)

This project runs on **Next.js (App Router)**.

## Development

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run start` — run the production server

## Routing

Routes live in the `app/` directory:

- `app/page.jsx` → `/`
- `app/kontakt/page.jsx` → `/kontakt`
- etc.

Most pages render existing UI from `src/views/` and `src/components/`.
